Instroduce 插件介绍

jQuery Show Hide is a simple jquery plugin that specify the selector to show or hide in some events such as click,change,blur and so on.

jQuery Show Hide 是一款简易的jquery插件，你可以在一些事件中，例如click,change,blur事件中指定容器的显示或隐藏

Structue文档结构

assets,dist------just some css,js files of this demo,not core.只是些用于此插件演示的css,js文件，不是核心文件

ShowHide-----core files 插件的核心文件

jquery-1.10.1.min.js------of course you won't forget 你不会忘记的jquery文件

Options 插件配置

Option Description
eventType event type such as click,change,blur keydown and so on.
事件类型 如click,change,blur keydown等
propKey attribute,such as value,text,checked,name,id,class event type such as click,change,blur keydown and so on.
元素属性 如value,text,checked,name,id,class等
propValue the value of the attribute, such as checked,true.
元素的属性值，例如chekced,true,等
autoToggle autoToggle=true,the selector will auto toggle in the specified event,at the same time,the 'propKey','propValue' is disable, but if autoToggle=false,the 'propKey','propValue' will enable
是否自动切换显示，隐藏.当autotoggle为true时 propKey,propValue设置无效
target the selector that to show or hide.But no more than 2 elements.
需要显示，隐藏的选择器，最多可设置两个元素
callEvent callback after the event triggered
当触发事件后的回调函数
